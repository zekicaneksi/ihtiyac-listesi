# frontend


## Used Technologies/Frameworks

- React
- Next.js
- TypeScript
- Native WebSocket
- Tailwind

## Prerequisites

- Node.js (20) with NPM


## Development

To install node modules

```sh
npm install
```

To run the project in development mode

```sh
npm run dev
```

## Deployment

To build the project

```sh
npm run build
```

To run the project in production mode

```sh
npm run start
```
